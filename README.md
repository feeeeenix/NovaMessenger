# NovaMessenger

Полноценный мессенджер-каркас с регистрацией, подтверждением email-кодом, входом, чатами, войсами, файлами, онлайн-статусом, уведомлениями и админ-доступом только для владельца.

## Запуск

```bash
npm install
npm start
```

Открой: `http://localhost:3000`

## Настройка почты

Скопируй `.env.example` в `.env` и заполни SMTP-данные.

## Владелец

По умолчанию:
- email: `owner@novamessenger.app`
- пароль: `123456`
